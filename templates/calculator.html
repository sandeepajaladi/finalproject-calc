Calculator

In ISO -601, Web development , we learnt how to develop website using bootstrap . We also learnt programming with Python and working with Flask.Flask is a micro web framework written in Python .Here is our final project .The user should be able to input at least two numbers into a web form and select one of the arithmetic operations (addition, subtraction, multiplication, or division) The user information should be written to a CSV file when the form is submitted The user should be shown a results page that shows all of the calculations stored in the CSV file printed as a table There should be validation for empty values for the input fields
